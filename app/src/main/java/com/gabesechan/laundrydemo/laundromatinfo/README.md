Laundromatinfo is a bit of a catch all.  It provides network access to the available pickup delivery times, and the item prices.  The item price API is likely solid, the pickup and delivery times are in a state of flux-  it was coded to immediate needs for a POC rather than careful decisions of the best way to do it, and the server is not yet sending real times and dates (just hardcoded relative times and dates).  As such no real effort has gone into improving that part until we have solid requirements.
